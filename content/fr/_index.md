## Bienvenue sur Mon Site

C'est la page d'accueil en français. Vous pouvez utiliser le sélecteur de langue dans l'en-tête (icône de globe) pour passer à l'anglais.

Le site supporte maintenant plusieurs langues avec des traductions complètes tout en préservant:
- Les termes techniques et le code
- Les noms et les dates
- Les noms de produits
- La terminologie de programmation

N'hésitez pas à créer vos propres traductions dans le répertoire `content/fr/` en suivant la même structure.
